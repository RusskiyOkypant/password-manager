# Менеджер паролей

Кроссплатформенный менеджер паролей с шифрованием и синхронизацией через Яндекс.Диск.

## Возможности

- Шифрование паролей с помощью Fernet (AES-128 + HMAC) и PBKDF2 с 600 000 итераций.
- История паролей (до 5 версий).
- Отложенная замена пароля (подтверждение после вставки на сайте).
- Автоматическая синхронизация с Яндекс.Диском.
- Автоблокировка через 5 минут бездействия.
- Встроенные уведомления в интерфейсе.
- Поддержка логинов и заметок для каждого сервиса.
- Экспорт и импорт данных в JSON и CSV.
- Поиск по сервисам.
- Генерация паролей с настраиваемыми параметрами.
- Десктопная версия (Tkinter) и мобильная (Kivy).

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости:

pip install -r requirements.txt

3. Создайте файл `.env` в папке `data/` и добавьте токен Яндекс.Диска:

YANDEX_TOKEN=ваш_токен

## Запуск

- Tkinter: `python run_tkinter.py`
- Kivy: `python run_kivy.py`

## Безопасность

- Мастер-пароль не сохраняется, используется только для генерации ключа.
- Данные шифруются локально, передаётся только зашифрованный файл.
- Синхронизация через облако не раскрывает содержимое.
