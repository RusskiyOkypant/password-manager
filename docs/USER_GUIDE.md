# Руководство пользователя

## Первый запуск

При первом запуске создаётся новое хранилище. Введите мастер-пароль и подтвердите его. Этот пароль будет использоваться для доступа ко всем данным.

## Главный экран

- **Список сервисов** слева. Нажмите на сервис, чтобы увидеть детали.
- **Правая панель**: отображает пароль, логин, ссылку, заметки.
- **Кнопки**: сгенерировать новый пароль (временный), установить свой пароль, история паролей, изменить ссылку, логин, заметки, удалить сервис.

## Добавление сервиса

Нажмите кнопку "+ Добавить сервис". Заполните поля, пароль можно оставить пустым для автоматической генерации.

## Смена пароля

- Нажмите "Сгенерировать новый" или введите свой пароль и нажмите "Установить".
- Пароль будет скопирован в буфер обмена.
- После вставки на сайте нажмите "Подтвердить смену" в менеджере.
- Если что-то пошло не так, нажмите "Отменить" — старый пароль останется.

## История паролей

Для каждого сервиса хранится до 5 последних паролей. Вы можете просмотреть историю, скопировать любой из них.

## Синхронизация

- При наличии интернета и валидного токена менеджер автоматически синхронизируется с Яндекс.Диском.
- Вручную синхронизировать можно через кнопку "Синхронизация" на главном экране.

## Автоблокировка

Приложение автоматически блокируется через 5 минут бездействия. Введите мастер-пароль для разблокировки.

## Экспорт/импорт

Функции доступны в интерфейсе (кнопки в главном меню или в настройках). Можно экспортировать данные в JSON (полный бэкап) или CSV (для просмотра в Excel).
